Rural Connect: Operational Performance & Business Intelligence Dashboard

 Project Overview
"Rural Connect" is an on-demand service platform designed to bridge the gap between skilled local workers (such as construction masonic workers, appliance technicians, and harvesting teams) and regional consumer demand across 7 key zones in Kolkata. 

This project establishes a complete data pipeline—from writing structured SQL schemas and solving transactional database errors to building a dynamic, interactive Power BI dashboard that delivers a 360-degree view of ecosystem health.



Key Insights & Dashboard Metrics
Total Sales: 409K generated on the platform, heavily anchored by core infrastructure services like *Rajmistry* (83K) and *Maid* services (55K).
Operational Growth: Bookings successfully scaled from baseline figures in January to a peak of 125 unique bookings by June.
Efficiency Trackers: Built row-level DAX aggregations displaying a **97% Service Fulfillment Rate** alongside a swift **6-hour average job turnaround time**.
Geographic Trends: Highlighted macro demand variations, showcasing high agricultural harvesting concentrations in Singur compared to heavy appliance servicing needs in tech corridors like Salt Lake Sector 5.

 Tech Stack & Tools Used
* **Database Management:** SQL Server (T-SQL) for database schema creation, constraint validation, data cleaning, and data integrity handling.
* **Business Intelligence:** Power BI Desktop for data modeling, layout UI design, and reporting.
* **Analytical Calculations:** DAX (Data Analysis Expressions) for dynamic performance metrics.



 Project Structure & Files in This Repo

Future Scope Roadmap
1. **AI Chatbot Dispatch:** Integrating an NLP chatbot to instantly parse regional language service requests and dynamically route jobs to nearby active workers.
2. **Predictive Capacity Planning:** Transitioning historical line charts into forward-looking advanced booking models to anticipate labor shortages.
3. **Multi-Work Bundle Engines:** Creating custom data models for cross-selling bundled services (e.g., *AC Service + Room Cleaning combos*).
